# NMIXX 'MIXX POP' Quantitative Analyzer

### Project Overview
This project aims to quantify the structural complexity of K-POP tracks, specifically focusing on **NMIXX's 'MIXX POP' genre**. By converting abstract musical variations into objective data, this module calculates the **'MIXX Index'**, a custom metric designed to measure the intensity of structural changes.
